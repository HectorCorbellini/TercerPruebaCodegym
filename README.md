Nivel 20, Lección 6
Tarea: escribe una pequeña búsqueda de texto. La pregunta en cada paso siguiente depende de la respuesta anterior.
Requisitos:

Debe ser un proyecto Maven utilizando las siguientes tecnologías: servlets, jsp, jstl .
Tomcat 9 se usará para el lanzamiento durante las pruebas , por lo que le aconsejo que lo use también durante el desarrollo.
Después de ganar o perder, debería ser posible reiniciar el juego.
Parte de la información debe almacenarse en la sesión. Puede ser el nombre del jugador, la cantidad de juegos jugados o cualquier otra cosa que elijas.

La lógica debe ser cubierta por pruebas. Utilice JUnit.
Requisito opcional:agregue una página de bienvenida en la que escribir un texto de bienvenida que cuente los antecedentes de este juego de búsqueda de texto.

Cargue el proyecto completo en su repositorio de GitHub .
